Problem solved today:

✅ Fruit Into Baskets (LeetCode 904)

Key idea:

This problem is essentially about finding the longest subarray containing at most 2 distinct elements.

Used a variable sliding window with a hashmap to track fruit frequencies.

When the window contains more than 2 fruit types, shrink the window from the left until the condition is satisfied again.

What this problem reinforced:

Variable sliding window works well when a constraint controls the window size.

Hash maps help maintain counts of elements inside the window.

Expand → check condition → shrink → update answer.
